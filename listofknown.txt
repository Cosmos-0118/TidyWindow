üîÅ Quick safe workflow (PowerShell): backup, disable, revert easily

Run as admin.

1) Export current service config (backup):

# backup current start types
Get-Service | Select-Object Name, Status, @{n='StartType';e={(Get-CimInstance Win32_Service -Filter "Name='$_'").StartMode}} |
  Export-Csv C:\service-backup.csv -NoTypeInformation


2) Disable a service safely (example SysMain):

# set to Manual first for testing
Set-Service -Name SysMain -StartupType Manual
# if all good after reboot, set Disabled:
Set-Service -Name SysMain -StartupType Disabled


3) Disable multiple safe services (example list):

$safe = @('GamingServices','GamingServicesNet','XboxGipSvc','XboxNetApiSvc','XblGameSave','GameBar','WerSvc','DiagTrack','RemoteRegistry','SysMain')
foreach($s in $safe){
    try{
        Set-Service -Name $s -StartupType Disabled -ErrorAction Stop
    } catch {
        Write-Warning "Could not change $s: $_"
    }
}


4) Revert from backup:

Import-Csv C:\service-backup.csv | ForEach-Object {
  $name = $_.Name
  $mode = $_.StartType
  if ($mode -in @('Auto','Automatic')) { Set-Service -Name $name -StartupType Automatic }
  elseif ($mode -in @('Manual')) { Set-Service -Name $name -StartupType Manual }
  elseif ($mode -in @('Disabled')) { Set-Service -Name $name -StartupType Disabled }
}
